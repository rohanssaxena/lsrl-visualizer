# Data input format

## Table layout

Each row is one observation:

| Column | Meaning |
|--------|---------|
| **x**  | Independent variable (any real number). |
| **y**  | Dependent variable (any real number). |

Empty rows are ignored when you click **Plot**. You can have up to **10** points.

## Plotting

- Axes include **zero** when sensible so the chart feels like a four-quadrant plane.
- **Pan / zoom** on the chart: use the mouse wheel and drag on the plot area.
- Hover **points** or the **regression line** for tooltips.

## Export mindset

There is no file import in this demo: values are typed directly. For reproducibility, keep a copy of your numbers elsewhere if you need them later.
